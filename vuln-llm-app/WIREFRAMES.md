# Wireframes (Conceptual)

## Catalog (/challenges)
- Header: title, progress, badges
- Grid of vulnerabilities (LLM01–LLM10)
- Each shows levels (low→impossible), availability badge

## Detail (/challenge/<id>)
- Challenge title, level, description
- Textarea for attempt; submit button
- Feedback panel; hint button (gated)
- Score + indicators; next steps

## Progress (/progress)
- Badges; per-vulnerability scores
- Recently solved; time-to-solve stats
- Locked/Unlocked levels
